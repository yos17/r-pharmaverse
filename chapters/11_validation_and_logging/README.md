## The Problem Without Logging

Run your ADSL program:

```r
source("pharm001_ch05_adsl.R")
```

It produces output. But you have no record of:
- When it ran
- What R version was used
- What package versions were used
- Whether any warnings were generated
- Whether it completed successfully

A SAS log file captures all of this automatically. In R, you add it explicitly with `logrx`.

## SAS Log vs. R Log: The Equivalence

In SAS, every program run automatically produces a `.log` file. In R, you add logging explicitly.

**In SAS (automatic):**
```
/* SAS .log file is created automatically — captures: */
/* - All NOTE:, WARNING:, ERROR: messages              */
/* - Code that was executed                            */
/* - Date/time of run                                  */
/* - SAS version                                       */
/* - Macro variable values (with MPRINT/SYMBOLGEN)     */
```

**Example SAS log messages:**
```
NOTE: PROCEDURE SORT used (Total process time):
      real time           0.01 seconds

NOTE: The data set WORK.ADSL has 254 observations and 32 variables.

WARNING: Variable RFXENDTC has 3 observations with missing values.
```

**In R (pharmaverse — logrx):**

`logrx::axecute()` runs a program and writes a `.log` file alongside it:

```r
library(logrx)

logrx::axecute("pharm001_ch05_adsl.R")
```

This runs the program and writes `pharm001_ch05_adsl.log` (by default, same directory). The log captures:

```
-----------------------------------------------------------------------------------------
Log file: pharm001_ch05_adsl.R
Run date and time: 2026-04-02 14:32:01 CEST
Username: yosia
Filepath: /projects/pharm001/programs/adam/pharm001_ch05_adsl.R
-----------------------------------------------------------------------------------------

--- R Version ---
R version 4.4.2 (2024-10-31)

--- Packages Attached ---
admiral_1.1.0, dplyr_1.1.4, lubridate_1.9.3 ...

--- Messages ---
[none]

--- Warnings ---
[none]

--- Program Status ---
Success
-----------------------------------------------------------------------------------------
```

## Inline Logging: SAS %PUT vs. R log_print()

**In SAS:**
```sas
/* Inline status messages in SAS log */
%PUT NOTE: === PHARM-001 ADSL Derivation ===;
%PUT NOTE: Run started: &SYSDATE9. &SYSTIME;

DATA adsl;
  SET dm(WHERE=(ACTARM NE "Screen Failure"));
RUN;
%PUT NOTE: Input DM: &SYSNOBS observations;

%PUT NOTE: === Derivation complete ===;
```

**In R (pharmaverse — logrx):**

Add `log_print()` inside programs for structured checkpoints:

```r
library(logrx)

log_print("=== PHARM-001 ADSL Derivation ===")
log_print(sprintf("Run started: %s", format(Sys.time())))

dm <- pharmaversesdtm::dm %>% filter(ACTARM != "Screen Failure")
log_print(sprintf("Input DM: %d rows, %d subjects",
                  nrow(dm), n_distinct(dm$USUBJID)))

# ... derivation ...

log_print(sprintf("ADSL output: %d rows", nrow(adsl)))
log_print(sprintf("SAFFL=Y: %d", sum(adsl$SAFFL == "Y", na.rm = TRUE)))
log_print("=== Derivation complete ===")
```

These messages appear in the log and in the console.

## What the Log Catches

Let's see what the log captures when something goes wrong. Add a deliberate warning to the ADSL program:

```r
# In pharm001_ch05_adsl.R, add this temporarily:
warning("TRTSDT missing for 3 subjects — check RFXSTDTC")
```

**In SAS**, this would appear in the log as:
```
WARNING: TRTSDT missing for 3 subjects — check RFXSTDTC
```

Run via `axecute()`. The `.log` file will contain:

```
--- Warnings ---
Warning message: TRTSDT missing for 3 subjects — check RFXSTDTC
```

Now you have a permanent record. When someone audits the run months later, the log shows the warning was generated and (in a real program) how it was addressed.

## Macro Debugging in SAS vs. R

**In SAS:**
```sas
/* Enable verbose SAS macro debugging */
OPTIONS MPRINT MLOGIC SYMBOLGEN;
/* MPRINT: shows code generated by macros */
/* MLOGIC: shows macro logic (IF/THEN, loops) */
/* SYMBOLGEN: shows macro variable substitutions */

%PUT &=myvar;      /* print macro variable value to log */
%PUT _ALL_;        /* print ALL macro variables */
```

**In R:**
```r
# Enable R debugging options
options(error = recover)      # after error: interactive call stack navigation
options(warn = 2)             # turn all warnings into errors (strict mode)

# Print values to console (equivalent to %PUT)
cat(sprintf("myvar = %s\n", myvar))
print(myvar)

# List all objects in current environment (equivalent to %PUT _ALL_)
ls()

# See option values
getOption("warn")
```

| SAS | R |
|-----|---|
| `%PUT NOTE: &myvar;` | `cat(myvar, "\n")` or `print(myvar)` |
| `%PUT _ALL_;` | `ls()` in `browser()` |
| `OPTIONS MPRINT;` | `options(error=recover)` |
| Automatic `.log` file | `logrx::axecute()` |
| `NOTE:` in log | `log_print()` |

## Updating Every Program

The pattern is the same for every PHARM-001 program. Here's the updated ADSL:

```r
# pharm001_ch11_adsl.R
# Study PHARM-001 — Chapter 11
# ADSL derivation with logging
# Replaces: pharm001_ch05_adsl.R

library(admiral)
library(pharmaversesdtm)
library(pharmaverseadam)
library(labelled)
library(dplyr)
library(stringr)
library(logrx)

log_print("=== PHARM-001 ADSL — START ===")

# ---- Inputs ----
dm  <- pharmaversesdtm::dm
ds  <- pharmaversesdtm::ds
ex  <- pharmaversesdtm::ex
sc  <- pharmaversesdtm::sc

log_print(sprintf("DM: %d rows", nrow(dm)))
log_print(sprintf("EX: %d rows", nrow(ex)))
log_print(sprintf("DS: %d rows", nrow(ds)))

# ---- Base ----
adsl <- dm %>%
  filter(ACTARM != "Screen Failure") %>%
  mutate(
    TRT01P  = ACTARM, TRT01A = ACTARM,
    TRT01PN = as.integer(factor(ACTARM, levels = sort(unique(ACTARM)))),
    TRT01AN = TRT01PN
  )

log_print(sprintf("After screen failure filter: %d subjects", nrow(adsl)))

# ---- Dates ----
adsl <- adsl %>%
  derive_vars_dt(new_vars_prefix = "TRTS",  dtc = RFXSTDTC, date_imputation = "first") %>%
  derive_vars_dt(new_vars_prefix = "TRTE",  dtc = RFXENDTC,  date_imputation = "last") %>%
  derive_vars_dt(new_vars_prefix = "DTH",   dtc = DTHDTC,    date_imputation = "first") %>%
  mutate(
    TRTDURD = as.integer(TRTEDT - TRTSDT + 1),
    DTHFL   = if_else(!is.na(DTHDT), "Y", NA_character_)
  )

n_trtsdt_na <- sum(is.na(adsl$TRTSDT))
if (n_trtsdt_na > 0)
  log_print(sprintf("NOTE: %d enrolled subjects have missing TRTSDT", n_trtsdt_na))

# ---- Demographics subgroups ----
adsl <- adsl %>%
  mutate(
    AGEGR1  = case_when(
      AGE < 40              ~ "<40",
      AGE >= 40 & AGE < 65  ~ "40-64",
      AGE >= 65             ~ ">=65",
      TRUE                  ~ NA_character_
    ),
    AGEGR1N = case_when(AGEGR1 == "<40" ~ 1L, AGEGR1 == "40-64" ~ 2L, AGEGR1 == ">=65" ~ 3L),
    SEXN    = case_when(SEX == "M" ~ 1L, SEX == "F" ~ 2L),
    RACEN   = case_when(
      RACE == "WHITE"                     ~ 1L,
      RACE == "BLACK OR AFRICAN AMERICAN" ~ 2L,
      RACE == "ASIAN"                     ~ 3L,
      TRUE                                ~ 99L
    )
  )

# ---- Disposition ----
ds_last <- ds %>%
  arrange(USUBJID, DSSTDTC) %>%
  group_by(USUBJID) %>% slice_tail(n=1) %>% ungroup() %>%
  select(USUBJID, DSDECOD)

adsl <- adsl %>%
  left_join(ds_last, by = "USUBJID") %>%
  mutate(
    COMPLFL = if_else(DSDECOD == "COMPLETED", "Y", "N"),
    EOSSTT  = case_when(
      DSDECOD == "COMPLETED" ~ "COMPLETED",
      is.na(DSDECOD)         ~ "ONGOING",
      TRUE                    ~ "DISCONTINUED"
    )
  )

# ---- Analysis flags ----
ex_dosed <- ex %>%
  group_by(USUBJID) %>%
  summarise(DOSED = any(!is.na(EXSTDTC)), .groups = "drop")

adsl <- adsl %>%
  left_join(ex_dosed, by = "USUBJID") %>%
  mutate(
    ITTFL   = "Y",
    SAFFL   = if_else(DOSED == TRUE, "Y", "N"),
    PPROTFL = if_else(SAFFL == "Y" & COMPLFL == "Y", "Y", "N")
  ) %>%
  select(-DOSED, -DSDECOD)

# ---- QC ----
log_print(sprintf("ADSL rows: %d", nrow(adsl)))
for (fl in c("ITTFL","SAFFL","PPROTFL","COMPLFL")) {
  log_print(sprintf("%s=Y: %d", fl, sum(adsl[[fl]] == "Y", na.rm = TRUE)))
}

# Compare against reference
ref <- pharmaverseadam::adsl
if (nrow(adsl) != nrow(ref))
  warning(sprintf("Row count mismatch: mine=%d ref=%d", nrow(adsl), nrow(ref)))
else
  log_print("Row count matches reference")

log_print("=== PHARM-001 ADSL — COMPLETE ===")
```

Run via `axecute()`:

```r
dir.create("logs", showWarnings = FALSE)
logrx::axecute("pharm001_ch11_adsl.R",
               log = "logs/pharm001_adsl.log")
```

## Package Version Management with renv

In SAS, reproducibility depends on the SAS version and installed macro libraries. In R, `renv` does the equivalent:

**SAS equivalent:**
```
/* SAS version control is implicit:
   - SAS version number is in every log
   - Macro libraries are catalogued by version
   - No equivalent to renv.lock for PROC/macro reproducibility */
```

**In R (pharmaverse — renv):**
```r
library(renv)

renv::init()      # first time: creates renv.lock
renv::snapshot()  # update renv.lock to current packages
renv::restore()   # on new machine: install locked versions
renv::status()    # check for drifts
```

Commit `renv.lock` to git. Anyone running `renv::restore()` gets identical package versions — reproducible across machines, across time.

## QC with diffdf — The PROC COMPARE Equivalent

**In SAS:**
```sas
/* Double-programming QC: compare two independent derivations */
PROC COMPARE DATA=adsl_mine COMPARE=adsl_qc LISTALL LISTVAR;
  ID USUBJID;
  /* LISTALL: list all matching and non-matching obs */
  /* LISTVAR: list all variables with differences */
  /* CRITERION=1E-6: tolerance for numeric comparisons */
RUN;
/* OUTPUT: summary table of differences by variable */
```

**In R (pharmaverse — diffdf):**
```r
library(diffdf)

diff_result <- diffdf(
  base      = adsl_mine,
  compare   = adsl_qc,
  keys      = "USUBJID",
  tolerance = 1e-6    # like PROC COMPARE CRITERION=1E-6
)

if (diffdf_issame(diff_result)) {
  cat("✓ Datasets identical\n")
} else {
  print(diff_result)
}
```

`diffdf` tells you exactly which rows and variables differ. Standard QC process: fix discrepancies, re-run, repeat until `diffdf_issame()` returns `TRUE`.

The output format is different but the information is equivalent:

| `PROC COMPARE` output | `diffdf` equivalent |
|-----------------------|---------------------|
| `Values Compared Equal` | `diffdf_issame()` returns `TRUE` |
| `Obs with differences for VAR` | `diff_result$VarDiff_VAR` |
| `Comparison Statistics` table | `diff_result$NumDiff` |
| `CRITERION=` option | `tolerance=` argument |

### Exercise 4

Write a `run_all.R` seed using `axecute()` for every program in order.

```r
# SAS equivalent: master run_all.sas using %INCLUDE for each program.
# logrx::axecute() is the R equivalent of %INCLUDE + automatic .log generation.

library(logrx)

# ---- Seed for run_all.R (Chapter 12 will extend this) ----

# Config
PROG_DIR <- "programs"
LOG_DIR  <- "logs"

dir.create(LOG_DIR, showWarnings = FALSE, recursive = TRUE)

# Pipeline order (must match SDTM → ADaM → TLG dependency chain)
# SAS equivalent:
# %INCLUDE "programs/sdtm/validate.sas";
# %INCLUDE "programs/adam/adsl.sas";
# ... etc.

programs <- list(
  list(path = file.path(PROG_DIR, "pharm001_ch02_validate.R"), label = "SDTM Validation"),
  list(path = file.path(PROG_DIR, "pharm001_ch03_dm.R"),       label = "SDTM DM Mapping"),
  list(path = file.path(PROG_DIR, "pharm001_ch11_adsl.R"),     label = "ADaM ADSL"),
  list(path = file.path(PROG_DIR, "pharm001_ch06_adae.R"),     label = "ADaM ADAE"),
  list(path = file.path(PROG_DIR, "pharm001_ch09_export.R"),   label = "XPT Export"),
  list(path = file.path(PROG_DIR, "pharm001_ch07_t14_1_1.R"),  label = "Table 14.1.1"),
  list(path = file.path(PROG_DIR, "pharm001_ch08_t14_3_1.R"),  label = "Table 14.3.1")
)

cat("=== PHARM-001 Run All (seed) ===\n")
cat(sprintf("Programs to run: %d\n\n", length(programs)))

for (prog in programs) {
  prog_name <- tools::file_path_sans_ext(basename(prog$path))
  log_file  <- file.path(LOG_DIR, paste0(prog_name, ".log"))

  if (!file.exists(prog$path)) {
    cat(sprintf("  [SKIP] %s — file not found: %s\n", prog$label, prog$path))
    next
  }

  cat(sprintf("  Running: %s ... ", prog$label))
  result <- tryCatch({
    logrx::axecute(prog$path, log = log_file)
    cat("OK\n")
    TRUE
  }, error = function(e) {
    cat(sprintf("FAIL: %s\n", conditionMessage(e)))
    FALSE
  })
}

cat("\nAll programs attempted. Check logs/ for details.\n")
cat("See Chapter 12 for the complete run_all.R with dependency checks.\n")
```
